### 简介

1. 接近APP的使用体验
2. 还算不错的细节和优化
3. 作者持续改进和处理反馈

视频页：双击全屏按钮竖屏播放，左右滑动切换侧边栏

搜索页：双击搜索按钮清空输入框，左右滑动切换分类

个人空间：双击搜索按钮全局搜索，左右滑动切换分类

Firefox 推荐扩展：[User Agent Switcher](https://addons.mozilla.org/zh-CN/firefox/addon/uaswitcher/)
